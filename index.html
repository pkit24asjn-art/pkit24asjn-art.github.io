Full stack development is the process of developing both the frontend and backend of applications. Any application has a frontend (user-facing) component and a backend (database and logic) component. The frontend contains the user interface and code related to user interactions with the application. The backend contains all the code required for the application to run, including integrations with data systems, communicating with other applications, and processing data.

üìÑ HTML (HyperText Markup Language)

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Example:

<h1>Hello, World!</h1>
<p>This is a simple paragraph in HTML.</p>

üé® CSS (Cascading Style Sheets)

CSS is Cascading style sheet which is used to style the web pages. It helps to make the web pages wonderful and makes user experience amazing. It enhances the presentation of html elements by applying styles like color, layout and spacing, CSS makes web pages visually appealing and responsive to various screen sizes. CSS are written inside the style tag of HTML document.

 What CSS does:

-Controls colors (text, background, borders, etc.)

-Changes fonts (size, style, weight)

-Manages layout (margins, padding, positioning, flexbox, grid)

-Adds effects (hover effects, animations, transitions)

-Makes websites responsive (adapt to mobile, tablet, desktop screens)
      
h1 {
  color: blue;
  text-align: center;
}
p {
  font-size: 18px;
  color: gray;
}

üöÄ Bootstrap

Bootstrap is a popular framework in HTML, CSS, and JavaScript for developing the different websites. It provides a collection of tools and components that help developers create, modern, visually appealing web pages quickly and efficiently. Bootstrap framework is well known for making responsive websites with ease. It makes sure your web project adjust fluidly to different screen and gadgets, giving users the best possible experience. It provides an extensive library of pre-made component which include navigation bars, buttons, forms and so on. The frame work can be modified by developers to meet the unique functional and design specifications of their projects. Bootstrap prioritizes accessibility, which makes it simpler to develop websites that are accessible to users with disabilities.

Example:

<button class="btn btn-primary">Bootstrap Button</button>
